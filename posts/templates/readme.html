ReadMe

This is a blog made with Django web framework to showcase WebGL games made with Unity3D game engine.

The game files are uploaded in zip format and are extracted with celery using redis as a message broker.

You can signup to rate and comment games but normal users can't upload games. If you want to be able to do that, contact me.

There is also a REST api for normal users to see the list of games and rate and comment them.

Libraries used

  • Django
  • Django REST framework
  • Celery
  • redis
  • and more!

Check the site's footer for source code and my FastAPI work sample!
